# 🧠 The Vat-Brain Project (Аквариум Савченко)
**Anti-Chatbot AI Assistant for Pure Code Generation**

### 📖 Идея (Sci-Fi Концепт)
Проект вдохновлен научно-фантастической повестью Владимира Савченко «Открытие себя» (1967). В книге ученый выращивает в изолированном аквариуме клона своего мозга, который лишен эмоций и умения "болтать", но обладает гипертрофированными способностями к чистой логике.

### 🎯 Проблема
Современные большие языковые модели (LLM), такие как ChatGPT или Claude, слишком "вежливы". При интеграции в среды разработки (IDE) программистов раздражает, когда на просьбу поправить строчку кода ИИ выдает абзац текста: *"Конечно! Я нашел ошибку. Вот ваш исправленный код: ..."*. Это тратит токены, время генерации и внимание пользователя.

### 💡 Решение
Мы взяли ультра-легкую модель **Qwen-2.5-1.5B** и с помощью технологии **LoRA (Low-Rank Adaptation)** "выжгли" из нее формат диалога. 
Это больше не чат-бот. Это строгая математическая функция. Вы даете ей сломанный код — она возвращает **только** исправленный синтаксис. Никаких приветствий, извинений и пояснений.

### 🛠 Технический стек и метрики
* **Base Model:** `Qwen/Qwen2.5-1.5B-Instruct`
* **Technique:** SFT (Supervised Fine-Tuning) via Unsloth
* **Hardware:** Обучено на 1x NVIDIA T4 (Kaggle) за 3 минуты.
* **VRAM Usage:** Всего 1.9 GB! Идеально для работы локально на ноутбуках без интернета.
* **Adapter Size:** ~50 MB.
